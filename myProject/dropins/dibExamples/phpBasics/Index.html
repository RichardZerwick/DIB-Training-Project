Getting a grip on PHP

NOTE: Copy all files in this folder to eg /www/phpBasics
Then navigate to http://localhost/phpBasics/Index.html

The purpose of this tutorial is to cover all the the most commonly used PHP constructs and functions in an easy to understand manner.
To receive maximum benefit, adjust and play around with the code until it is fully understood.
Grasping the concepts should provide developers with a firm foundation to write PHP scripts.

The files below are all executable .php files.
Install a PHP code editor, eg. Visual Studio Code (with the PHP Extension Pack) or PHP CodeLobster, to study their contents.
Also activate the PHP Debugger in the editor to allow you to step through code and view values as it executes.
